# Inti AI Project

Official public overview of the Inti ecosystem.

This repository provides a clear, timestamped public record of Inti brand usage, product naming, and high-level direction.

## Mission

Inti builds practical human+AI systems for document intelligence, read-aloud learning, and collaborative creation.

## Product Family (High-Level)

- **Inti Docs** — AI-assisted document understanding, PDF reader workflows, and knowledge extraction.
- **Inti Docs Enterprise** — organizational deployment plus enterprise document-to-markdown conversion pathways.
- **Inti Reader** — read/listen/discuss experiences for deep comprehension.
- **Inti Author** — human+AI co-writing and structured creation workflows.
- **Inti Creator** — artistic human+AI collaborative creation.
- **Inti Code** — software development support and engineering workflows.
- **IntiNet** — long-term network layer for connected Inti systems.
- **Intellipedia** — structured knowledge layer within the broader Inti ecosystem.

> Note: Product scope, names, and sequencing may evolve.
