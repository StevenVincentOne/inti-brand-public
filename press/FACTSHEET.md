# Inti — Press Factsheet (v1)

## What is Inti?

Inti is a human+AI ecosystem focused on helping people and teams read, understand, convert, and create with documents and knowledge.

## Core Focus Areas

- Document intelligence and AI-assisted reading
- Read aloud and comprehension workflows
- PDF reader and document conversion pathways
- PDF to markdown conversion for practical reuse
- Human+AI co-creation

## Product Line (Current Public Names)

- Inti Docs
- Inti Docs Enterprise
- Inti Reader
- Inti Author
- Inti Creator
- Inti Code
- IntiNet
- Intellipedia
