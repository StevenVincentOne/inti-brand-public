# Inti Brand (v1)

## Canonical Brand

- **Master Brand:** Inti
- **Primary Domain:** `myinti.ai`

## Product Naming (Current)

- Inti Docs
- Inti Docs Enterprise
- Inti Reader
- Inti Author
- Inti Creator
- Inti Code
- IntiNet
- Intellipedia

## Naming Rules

1. Use **Inti** with capital “I” as the brand root.
2. Use product names in Title Case (e.g., **Inti Docs**, not `inti docs`).
3. Prefer `product.myinti.ai` style for official web properties.
4. Avoid unofficial abbreviations in public-facing material.

## Brand Positioning (Short)

Inti is a human+AI platform focused on practical knowledge, creation, and collaboration workflows.

## Visual Identity

Official logos and visual assets will be published under `/assets`.

## Public Source of Truth

Until superseded, this repository and links from `myinti.ai` define the public canonical naming set.
